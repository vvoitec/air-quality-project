# air-quality-project

## Project setup
Project needs .env.local file with token genereted from https://api.waqi.info
```
VUE_APP_WAQI_TOKEN=%YOUR TOKEN%
```
To install dependencies: 
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```
